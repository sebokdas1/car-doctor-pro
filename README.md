Install Dependencies:

```bash
npm i axios bcrypt mongodb next-auth react-toastify
```

Install daisyUI as a Node package:

```bash
npm i -D daisyui@latest
```

Add daisyUI to tailwind.config.js:

```bash
  plugins: [
    require('daisyui'),
  ],
```

Install React-Icons:

```bash
npm install react-icons --save
```
